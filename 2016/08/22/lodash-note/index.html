这里纪录lodash里面一些最常用函数的使用

lodash 已经算是我在任何项目都必须加入的东西，不管是前端还是后端项目，它带来的便利性很大，而且很方便，导致现在我用原生js写数组/字符串之类的已经是不会写了，这是好事还是坏事呢？

这里的文档是~4版本

按照文档中的顺序一个个来：

Array 数组操作

_.concat

连接数组，不改变原数组

_.indexOf

找下标，一般做有无判断

如果被找的是数组，直接 _.indexOf([1, 2, 3, 4, 5, 6], item) < 0，如果被找的是对象，匹配的是对象的值，用_.values搭配

记住 < 0 为 true 是没有

_.uniq

去重，初始化数据的时候可以用

Collection 集合(Array|Object)

_.filter

一般 _.filter(Boo) 用于去空，和 _.remove 不同点在于： _.remove 是改变原数组的，_.filter 不改变原数组，需要最后赋值给新数组

_.forEach

没有返回值

_.map

有返回值

_.reduce

_.sample

集合里面随机取一个值

_.shuffle

打乱顺序

_.sortBy
